This is the top!

I'm building this page to practice my front-end skills, and also to get used to using git and Github. I'm also learning bash while I'm at it. This is also the first time I've played around with Flexbox! Master branch commits get deployed automatically to spfict.com by deployHQ (what a cool tool!).

Vanilla JavaScript only!

Everything on this page uses simple HTML5, CSS, and plain JavaScript. No fancy libraries or frameworks (at least not yet).

Why am I shrinking?

This flex column shrinks 8 times as fast as the sidebar next to it.

Nice scroll effect, eh?

It get's the Rect of the target, subtracts the Rect of the body, then subtracts 55px for the sticky header and uses window.scrollTo with super smooth behavior. One little function and I can make it go anywhere by passing the right parameter.

The scroll anchors on the left use it. The disappearing, clickable scroll indicator above the fold uses it. Even the back to top link in the bottom right uses it.
